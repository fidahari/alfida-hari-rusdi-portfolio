# Demo Website OrangeHRM

Project ini adalah **demo website untuk keperluan testing**.  
Tujuannya adalah untuk memperlihatkan bagaimana sistem bekerja dan merespons berbagai interaksi dari user secara keseluruhan.  

## Tujuan Testing
- Memastikan website berfungsi sesuai harapan
- Memvalidasi interaksi user dengan sistem
- Menangani kondisi input yang berbeda-beda
- Menunjukkan perilaku sistem saat navigasi dilakukan

## Fokus
Testing difokuskan pada **pengalaman pengguna dan alur kerja secara umum**, mencakup seluruh fitur website tanpa masuk ke detail spesifik.

## Penggunaan
1. Buka website demo di browser (direkomendasikan Chrome, Windows 10)  
2. Coba berinteraksi dengan website secara normal  
3. Amati apakah sistem merespons sesuai harapan dan pesan validasi muncul bila diperlukan
